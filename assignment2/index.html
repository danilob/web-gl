Primeira Atividade: RASTERIZAÇÃO

(Assignment 02)


Descrição da atividade: Implemente o algoritmo de Bresenham, considerando todos os tipos de orientação de reta, a partir de dois pontos dados. Construa uma grid determinada a partir da criação de uma reta (que o usuário irá informar). Faça com que pelo layout o usuário possa alterar os valores dos pontos em tempo real.

Para realizar esta atividade se faz necessário o uso de Brasenham. Neste link você pode conferir os slides que foram utilizados como base para o desenvolvimento desta atividade, bem como a referência bibliográfica utilizada. O bônus da atividade seria implementar a rasterização de círculos e elipses.

Abaixo demonstro a aplicação do algoritmo de Brasenham para linhas. Você pode informar uma nova reta informando o seguinte código: drawLine(x0,y0,x1,y1);, onde (x0,y0) é o ponto inicial da reta e (x1,y1) é o ponto final da reta. Teste os seguintes valores, substituindo a linha relacionada no textarea: drawLine(-8,14,9,5), drawLine(-8,14,9,5), drawLine(30,15,25,5) e drawLine(20,19,15,3);
