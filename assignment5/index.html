Quinta Atividade: CONTROLE DE ORIENTAÇÃO

(Assignment 04)


Descrição da atividade: Utilize como base este tutorial para realizar o desenho de um cubo, a partir deste cubo realize as seguintes implementações: representar uma orientação com quatérnions para realizar a rotação do cubo interpolando entre dois quatérnions determinados utilizando Slerp. Tanto o tempo como os quatérnions deverão ser especificados pelo usuário.

No código abaixo você poderá determinar o quatérnio utilizando ângulos de Euler (ex.: q = new Quaternion(90,0,0)) e no frame adicionar o Quatérnion e o tempo, em segundos, de cada frame (ex.: f1 = new Frame(q,5)). Após definir todos os frames é utilizando uma lista que será responsável por realizar a interpolação entre os frames utilizando a função goAnimate(lista) e dando o play no botão.
